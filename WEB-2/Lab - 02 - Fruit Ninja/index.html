Fruit Ninja! (exemple)

Fonctionnement du jeu
  • Le fichier CSS et HTML sont déjà fournis!
  • Des fruits montent et redescendrent dans la page du jeu, ou encore passe de gauche à droite (ce qui est plus simple).
  • Il faut cliquer/couper les fruits, avant que ceux-ci ne soient plus visible dans l'écran
  • Pour chaque fruit « coupé », le pointage/score augmente de 1
  • Après 3 fruits qui n’ont pas été coupés, le jeu se termine
  • Lorsqu’un fruit est coupé, ses classes CSS changent (avant : "fruit fruit4", après : "fruit fruit4-cut"). Ce changement de classe permet de changer l'image.
  • Optionnel : Lorsque l’usager clique sur un fruit et qu’il le coupe, une image « slash » de l’épée apparait pour quelques tick (voir image sword-slash.png)
