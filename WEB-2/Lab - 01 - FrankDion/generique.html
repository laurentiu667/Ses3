Page 3 : Animation

  1. Déplacer le texte du générique très bas en utilisant les propriétés CSS display:fixed et top.
  2. Créer un fichier js/javascript.js et l'importer dans la page (<script src...)
  3. Dans le fichier JavaScript, créer une fonction JavaScript (ex: moveGeneric()) qui monte de 1 pixel le div du générique. Utilisez une variable globale qui contient la hauteur du générique (exemple : let currentTop = 600), que vous décrémenterez à chaque appel de fonction.
  4. Ajouter le code permettant d'attendre que la page soit chargée, puis appeler la fonction moveGeneric().
  5. Utiliser un setTimeout ou setInterval qui appelle la fonction moveGeneric() à chaque 30msec.
  6. Si l'utilisateur clique sur le div#generic, alors cela inverse le déplacement du texte (ex : s'il monte, alors il commence à descendre)
